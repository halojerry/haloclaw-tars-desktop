## 核心能力保留

### 浏览器自动化
- **核心实现**: `packages/agent-infra/browser-use/`
- **桌面集成**: `apps/ui-tars/src/main/ipcRoutes/browser.ts`
- **浏览器操作**: `multimodal/gui-agent/operator-browser/`
- **支持**: Chrome/Edge/Firefox + Puppeteer

### MCP集成
- **MCP客户端**: `packages/agent-infra/mcp-client/`
- **MCP Agent**: `multimodal/omni-tars/mcp-agent/`
- **MCP Servers**: `packages/agent-infra/mcp-servers/`

### GUI Agent
- **Agent SDK**: `@ui-tars/sdk`
- **本地操作**: NutJS (键盘/鼠标控制)
- **浏览器操作**: Puppeteer + DOM解析
