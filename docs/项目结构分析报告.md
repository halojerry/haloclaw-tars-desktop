## 3. 核心能力分析

### 3.1 浏览器自动化

**关键路径**: `packages/agent-infra/browser-use/`

| 组件 | 功能 |
|------|------|
| `context.ts` | 浏览器上下文管理 |
| `page.ts` | 页面操作 |
| `utils.ts` | 工具函数 |

**桌面集成**: `apps/ui-tars/src/main/ipcRoutes/browser.ts`

### 3.2 MCP集成

| 组件 | 路径 |
|------|------|
| MCP Client | `packages/agent-infra/mcp-client/` |
| MCP Agent | `multimodal/omni-tars/mcp-agent/` |
| MCP Servers | `packages/agent-infra/mcp-servers/` |

### 3.3 AI Operator

**本地操作**: `apps/ui-tars/src/main/agent/operator.ts`
- NutJSElectronOperator: 本地桌面控制
- 支持: click, type, scroll, drag, hotkey, wait

**Action Spaces**:
```
click(start_box='[x1,y1,x2,y2]')
left_double(start_box='[x1,y1,x2,y2]')
drag(start_box, end_box)
hotkey(key='')
type(content='')
scroll(direction='down/up/right/left')
wait()
finished()
call_user()
```
